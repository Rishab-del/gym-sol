# 🏋️ Gym App

A modern fitness tracking mobile application built with React Native, Expo, and Expo Router.

## 🚀 Features

### 🏠 Home Dashboard
- Daily calorie tracking
- Water intake monitoring
- Protein tracking
- Weight tracking
- Daily workout overview
- Modern dark UI

### 🍽 Food Tracker
- Track meals throughout the day
- Monitor calories
- Protein, carbs, and fat goals
- Daily nutrition summary

### 💪 Workout Tracker
- Exercise logging
- Push/Pull/Leg workout plans
- Sets & reps tracking
- Workout progress monitoring

### 👤 Profile
- User fitness statistics
- Goal management
- Weight tracking
- Progress overview
- Fitness streaks

## ⚙️ Installation

Clone the repository:

bash git clone https://github.com/Rishab-del/gym-sol.git 

Install dependencies:

bash npm install 

Run the project:

bash npx expo start
